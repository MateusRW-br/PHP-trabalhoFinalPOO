Trabalho Final de Programação Orientada a Objeto 

Conceito:
  Sistema gerenciador de tarefas desenvolvido em PHP, que permite o cadastro, edição, remoção e listagem de tarefas pessoais e profissionais. O projeto utiliza os princípios da Programação Orientada a Objetos, como encapsulamento, herança e polimorfismo, para modelar diferentes tipos de tarefas. O sistema também realiza o gerenciamento de memória por meio da criação e destruição de objetos durante a execução, além de possibilitar a organização das tarefas por prioridade, status e categoria.
